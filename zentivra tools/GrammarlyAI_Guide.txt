3. KEY FEATURES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✦ Grammar & Spelling Correction (Core Feature)
  Basic grammar mistakes automatically detect & fix చేస్తుంది.
  Typos, misspellings, homophones (there/their/they're) - all caught!

✦ Tone Detection
  మీరు రాసిన text యొక్క tone analyze చేస్తుంది.
  "Your email sounds a bit harsh" - warning ఇస్తుంది!
  Confident, Friendly, Formal, Casual - tone labels వస్తాయి.

✦ Clarity Score
  మీ sentences clear గా ఉన్నాయా? 0-100 score ఇస్తుంది.
  Complex sentences ని simplify చేయమని suggest చేస్తుంది.

✦ Conciseness
  Unnecessary words, redundant phrases identify చేస్తుంది.
  "Due to the fact that" → "Because" - shorter = clearer!

✦ Vocabulary Enhancement
  Repetitive words కి better alternatives suggest చేస్తుంది.
  "Good" ని "Excellent, Outstanding, Impressive" suggest చేస్తుంది.

✦ Plagiarism Checker (Premium)
  16 billion+ web pages తో compare చేస్తుంది.
  Students, journalists, researchers కి crucial!

✦ GrammarlyGO (AI Generation)
  "Write a professional apology email" - instant draft!
  "Rewrite this in formal tone" - complete transformation.
  "Make this shorter" - key points retain చేసి shorter.

✦ Personal Dictionary
  Company-specific terms, proper nouns add చేయవచ్చు.
  "Hyderabad" ని misspelling గా flag చేయకుండా.

✦ Writing Goals Setting
  Audience, Formality, Domain, Tone set చేయవచ్చు.
  AI suggestions accordingly adjust అవుతాయి.

5. STEP-BY-STEP GUIDE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Setup చేయడం
  → grammarly.com కి వెళ్ళండి
  → Free account create చేయండి
  → Browser extension install చేయండి (Chrome recommended)
  → Desktop app download చేయవచ్చు

STEP 2: Browser Extension - Everywhere Works!
  → Chrome extension install అయిన తర్వాత
  → Gmail open చేయండి - Grammarly icon కనిపిస్తుంది
  → Email రాస్తే automatically underlines చేస్తుంది
  → Click on underline → suggestion వస్తుంది

STEP 3: Writing Goals Set చేయండి
  → Grammarly editor లో "Set Goals" click చేయండి
  → Audience: Expert/Knowledgeable/General
  → Formality: Formal/Neutral/Informal
  → Domain: Academic/Business/Creative/General/Technical
  → These settings customize suggestions!

STEP 4: Grammarly Editor Use చేయడం
  → app.grammarly.com కి వెళ్ళండి
  → "New Document" create చేయండి
  → Text paste చేయండి లేదా రాయండి
  → Right side: All suggestions appear
  → Each card: Issue, explanation, fix option

STEP 5: GrammarlyGO (AI Writing)
  → Editor లో "AI" button click చేయండి
  → "What do you want to write?" prompt వస్తుంది
  → Instruction type చేయండి:
    "Draft a professional email requesting deadline extension"
    "Rewrite this paragraph to sound more confident"

STEP 6: Plagiarism Check (Premium)
  → Document complete అయిన తర్వాత
  → "Plagiarism" button click చేయండి
  → Wait for check (1-2 minutes)
  → Percentage + sources shown

STEP 7: Microsoft Word Integration
  → Grammarly for Windows install చేయండి
  → Word open చేస్తే automatically Grammarly panel appears
  → Word documents directly check చేయవచ్చు!

9. TIPS & TRICKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 TIP 1: Goals Always Set చేయండి
  Academic paper రాస్తుంటే - Expert audience, Formal, Academic domain.
  Friend కి email రాస్తుంటే - General audience, Informal, General.
  Wrong goals = wrong suggestions!

💡 TIP 2: Don't Accept Everything Blindly
  Grammarly suggest చేసిన అన్నీ correct కాదు. 
  Creative writing లో suggestions skip చేయవచ్చు.
  Always use judgment!

💡 TIP 3: "Learn" from Suggestions
  "Why is this wrong?" explanation read చేయండి.
  English grammar naturally improve అవుతుంది.

💡 TIP 4: Email Tone Check
  Important email పంపే ముందు Grammarly లో paste చేయండి.
  Tone "Confident" + "Friendly" = professional email!

💡 TIP 5: GrammarlyGO Power
  "Improve this email's opening line" - single sentence improve చేయవచ్చు
  "Make this more persuasive" - sales copy improve
  "Simplify this for a general audience" - jargon remove

💡 TIP 6: Personal Dictionary
  Telugu proper nouns (Vijayawada, Visakhapatnam, Tirupati) add చేయండి.
  Constant flagging avoid అవుతుంది.

💡 TIP 7: Weekly Writing Stats
  Grammarly weekly email ఇస్తుంది - your writing stats.
  "This week you were more productive than 90% of users"
  Progress track చేయవచ్చు!

💡 TIP 8: LinkedIn Optimization
  LinkedIn profile, posts రాస్తుంటే Grammarly automatically works.
  Professional impression create అవుతుంది.

💡 TIP 9: Keyboard Shortcut
  Ctrl+Alt+G (Windows) - quick Grammarly access
  Ctrl+Enter - accept suggestion

💡 TIP 10: Student Hack
  Assignment submit చేయే ముందు Grammarly లో ఒకసారి run చేయండి.
  Grammar marks తప్పకుండా improve అవుతాయి!
