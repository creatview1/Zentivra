Text-based editing - transcript edit = media edit.

Descript Products:
1. Transcription - Auto-transcribe with 95%+ accuracy
2. Text-based Editing - Document edit = video edit
3. Overdub - AI voice clone (text to your voice)

5. STEP-BY-STEP GUIDE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Account
  → descript.com కి వెళ్ళండి
  → Free account create చేయండి
  → Desktop app download చేయండి (Mac/Windows)

STEP 2: New Project
  → "New Project" click చేయండి
  → Video/Audio file import చేయండి
  → AI transcription starts automatically

STEP 3: Transcript Review
  → Transcription complete అయిన తర్వాత
  → Text review చేయండి - errors correct చేయండి
  → Speaker names assign చేయండి

STEP 4: Text-Based Edit
  → Unwanted sections text లో select చేయండి
  → Delete press చేయండి - video automatically cut అవుతుంది!
  → "Remove filler words" option try చేయండి

STEP 5: Studio Sound Apply
  → Audio track select చేయండి
  → "Effects" → "Studio Sound" ON చేయండి
  → Before/After compare చేయండి - dramatic improvement!

STEP 6: Overdub Setup
  → "Overdub" section → "Record your voice"
  → 10 sentences read చేయండి (provided script)
  → AI voice clone ready (30 minutes processing)

STEP 7: Correct with Overdub
  → Transcript లో wrong word select చేయండి
  → Correct word type చేయండి
  → Your AI voice automatically reads correction!

STEP 8: Add Captions
  → "Captions" → "Auto Captions"
  → Style customize చేయండి
  → Social media format export!

STEP 9: Export
  → "Export" → Video format select
  → Resolution, format choose
  → Done!
