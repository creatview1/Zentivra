STEP 1: Setup
  → copy.ai కి వెళ్ళండి
  → Free account create చేయండి
  → Workspace setup చేయండి (company name, business type)

STEP 2: Infobase Setup (Important!)
  → "Infobase" section కి వెళ్ళండి
  → Company description add చేయండి
  → Products/services describe చేయండి
  → Target audience define చేయండి
  → This info all future outputs లో use అవుతుంది!

STEP 3: Brand Voice Setup
  → Brand voice examples provide చేయండి
  → Tone: Professional/Casual/Friendly/Authoritative
  → AI matches your brand in all outputs

STEP 4: Tool Use
  → "New Project" create చేయండి
  → Chat లో request చేయండి: "Write 5 Facebook ad headlines for our organic food brand"
  → లేదా Tools browse చేయండి - specific tool select చేయండి

STEP 5: Refine
  → Multiple variations generate చేయండి
  → Best elements combine చేయండి
  → "Make it more urgent" "Shorter" "More casual"

STEP 6: Team Workflow
  → Invite team members
  → Assign projects
  → Review and approve content
