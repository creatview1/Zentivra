5. STEP-BY-STEP GUIDE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Account Create
  → canva.com కి వెళ్ళండి
  → Google account తో signup చేయండి (instant!)
  → Education account? Free Canva Pro!

STEP 2: Design Type Select
  → Search: "Instagram post" లేదా "YouTube thumbnail"
  → Size automatically set అవుతుంది!
  → Template browse చేయండి

STEP 3: Template Customize
  → Template click చేయండి - editor opens
  → Text click చేయండి → edit చేయండి
  → Colors, fonts change చేయండి
  → Images replace చేయండి

STEP 4: Magic Design Try చేయండి
  → canva.com home page లో "Magic Design" click చేయండి
  → Describe your design in English:
    "Colorful Instagram post for Diwali, festive theme, with text 'Happy Diwali'"
  → AI generates multiple options!
  → Best one customize చేయండి

STEP 5: Background Remove
  → Photo upload చేయండి
  → "Edit Photo" click చేయండి
  → "BG Remover" click చేయండి
  → Background disappears! Magic! ✨

STEP 6: Text to Image
  → Design లో "Apps" → "Text to Image" click చేయండి
  → Prompt type చేయండి
  → Style select చేయండి (Watercolor, 3D, Photo...)
  → Generate!

STEP 7: Download
  → "Share" button → "Download"
  → Format: PNG (high quality), JPEG, PDF, MP4 (videos)
  → Social media కి directly share option కూడా ఉంది!

💡 TIP 1: Keyboard Shortcuts
  C = Crop, T = Text, R = Rectangle, L = Line
  Ctrl+D = Duplicate element
  Ctrl+G = Group elements
  Time dramatically save అవుతుంది!

💡 TIP 2: Brand Colors Set చేయండి
  Brand Kit లో exact hex codes save చేయండి.
  All designs automatically consistent!

💡 TIP 3: Telugu Fonts
  Search: "Noto Sans Telugu" - Google font, free
  Telugu text perfectly renders!

💡 TIP 4: YouTube Thumbnail Formula
  Large face image + bright background + bold text 3 words
  CTR dramatically increase అవుతుంది!

💡 TIP 5: Magic Design Prompt Tips
  Specific describe చేయండి: "minimalist, blue tones, professional"
  10 options లో definitely good one వస్తుంది

💡 TIP 6: Templates Hack
  Pro template లో watermark ఉంటుంది.
  Similar free template search చేయండి - usually exists!

💡 TIP 7: Resize Feature (Pro)
  Instagram post → Facebook cover → Twitter header → one click resize!
  Multiple formats instantly.

💡 TIP 8: Canva Mobile App
  Phone camera తో photo click చేసి directly Canva app లో edit.
  Social media వెంటనే post చేయవచ్చు!
