5. STEP-BY-STEP GUIDE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Account
  → tome.app కి వెళ్ళండి
  → Google account తో login చేయండి
  → Free account immediately accessible

STEP 2: Create with AI
  → "Create" button click చేయండి
  → Prompt box లో type చేయండి:
    "Create a presentation about the future of AI in Indian education,
     10 pages, include statistics, modern design, blue theme"
  → "Generate" click చేయండి
  → 30-60 seconds: Complete presentation!

STEP 3: Review & Edit
  → Each page individually edit చేయవచ్చు
  → Text click → edit
  → Image click → change or regenerate
  → Page reorder, add, delete possible

STEP 4: AI Assist Individual Pages
  → Page select చేయండి
  → "AI" icon click చేయండి
  → "Add more statistics"
  → "Make this more concise"
  → Page specifically improves!

STEP 5: Embed Media
  → "+" icon → "Embed"
  → YouTube URL paste చేయండి - video embeds!
  → Figma design, Loom video, Google doc - all possible

STEP 6: Theme Customize
  → Right sidebar లో "Theme" click చేయండి
  → Color palette, font choose
  → All pages instantly update!

STEP 7: Share
  → "Share" button click చేయండి
  → "Copy link" - anyone can view
  → Password protect option
  → Email specific people
