STEP 1: Account Setup

  → Free account లో 150 tokens/day

STEP 2: Dashboard Explore
  → "AI Image Generation" click చేయండి

  → Center: Prompt input
  → Right side: Settings

STEP 3: Model Select చేయండి

  → Number of images: 1-8
  → Image dimensions: 512x512, 1024x1024 etc.
  → Guidance scale: Prompt follow చేసే intensity

  → "Generate" button click చేయండి
  → Images 20-40 seconds లో వస్తాయి

  → "AI Canvas" section కి వెళ్ళండి
