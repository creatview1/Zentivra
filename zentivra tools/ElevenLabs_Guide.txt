5. STEP-BY-STEP GUIDE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Account
  → elevenlabs.io కి వెళ్ళండి
  → Free account: 10,000 characters/month

STEP 2: Text to Speech
  → "Text to Speech" click చేయండి
  → Voice select చేయండి (pre-made voices browse చేయండి)
  → Text paste చేయండి
  → Settings adjust:
    Stability: 50% (natural variation)
    Clarity: 75% (clear pronunciation)
  → "Generate" click చేయండి
  → Play, then Download!

STEP 3: Telugu Text Try చేయండి
  → Same interface, Telugu text paste చేయండి
  → "Multilingual v2" model select చేయండి (Telugu support)
  → Voice choose చేయండి
  → Generate - Telugu voice వస్తుంది!

STEP 4: Voice Cloning
  → "Voice Lab" → "Add Generative Voice" → "Instant Voice Clone"
  → 30 seconds - 2 minutes audio upload చేయండి (clear recording)
  → Voice name give చేయండి
  → Your voice clone ready!
  → Now type any text - your voice speaks it!

STEP 5: Dubbing Studio
  → "Dubbing Studio" click చేయండి
  → Video file upload (YouTube URL paste చేయవచ్చు!)
  → Source language: Hindi
  → Target language: Telugu
  → "Dub" click చేయండి
  → 5-10 minutes processing
  → Telugu dubbed video download!

STEP 6: Projects (Audiobook)
  → "Projects" section click చేయండి
  → New project create చేయండి
  → Content paste (chapter by chapter)
  → Different voices assign చేయండి
  → Generate all chapters

💡 Voice Cloning Quality Tips:
  - Quiet room లో record చేయండి
  - Different sentences, emotions include చేయండి
  - 2-3 minutes audio = excellent clone

💡 SSML Tags Use చేయండి:
  <break time="1s"/> - pause add చేయడానికి
  <emphasis> word </emphasis> - stress a word
  Natural-sounding speech వస్తుంది!

💡 Telugu Pronunciation Fix:
  Difficult words ని phonetically spell చేయండి
  "Hyderabad" → "Hye-dra-baad"

💡 Free Tier Maximize:
  10,000 chars = carefully plan చేయండి
  Short test, then long generation

💡 Voice Selection:
  Content type బట్టి voice choose చేయండి:
  Educational: Calm, clear

  Story: Emotional, expressive
