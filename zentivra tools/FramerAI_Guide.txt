5. STEP-BY-STEP GUIDE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Account Create
  → framer.com కి వెళ్ళండి
  → "Get Started for Free" click చేయండి
  → Google account తో login

STEP 2: New Project
  → "New Site" click చేయండి
  → "Start with AI" option select చేయండి

STEP 3: AI Generation
  → Describe your website:
    "Portfolio website for a Hyderabad-based UI/UX designer.
     Dark theme, modern, minimal. Sections: Hero, Work, About, Contact.
     Professional, creative feel."
  → "Generate" click చేయండి
  → 3-4 style variations వస్తాయి!

STEP 4: Select & Customize
  → Favorite style select చేయండి
  → Editor opens - website already looks great!

STEP 5: Edit Content
  → Text click → edit మీ actual content
  → Images replace: Click image → "Replace" → upload
  → Colors: Right panel → Theme → customize

STEP 6: Add Animations
  → Element select చేయండి
  → Right panel → "Animate"
  → Scroll animation, hover effect choose
  → Preview లో see చేయండి - amazing!

STEP 7: Mobile View
  → Top bar లో device icons click చేయండి
  → Mobile layout check చేయండి
  → Adjustments make చేయండి

STEP 8: SEO Setup
  → Site Settings → SEO
  → Page title, description fill చేయండి
  → OG image (social share image) upload

STEP 9: Connect Domain & Publish
  → Site Settings → Custom Domain
  → Domain enter చేయండి, DNS configure
  → "Publish" click!

STEP 10: CMS Setup (for Blog)
  → "CMS" section
  → Collection create (Blog Posts)
  → Fields define (title, date, content, image)
  → Posts add చేయండి
  → Blog template connect

9. TIPS & TRICKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 TIP 1: AI Prompt - Detailed చేయండి
  "Dark themed SaaS landing page, bold typography, purple accent,
   animations, hero with dashboard mockup, pricing section, 
   testimonials, modern 2024 design" - specific output వస్తుంది.

💡 TIP 2: Multiple Generations
  Same prompt తో 3-4 times generate చేయండి.
  Each time different style varia ions వస్తాయి.

💡 TIP 3: Framer Template Gallery
  framer.com/templates - free templates available.
  Popular templates use చేసి customize చేయండి.
  Starting point గా excellent!

💡 TIP 4: Figma to Framer
  Figma design ఉంటే - Framer directly import చేయగలదు.
  "Figma Plugin: Framer" install చేయండి.
  Design automatically live website అవుతుంది!

💡 TIP 5: Animations - Don't Overdo
  Subtle animations = Professional
  Too many animations = Distracting
  Apple.com style: Minimal, purposeful animations.

💡 TIP 6: Performance Test
  Site publish అయిన తర్వాత PageSpeed Insights లో test చేయండి.
  Images optimize చేయండి - WebP format use.

💡 TIP 7: CMS for Blog
  Blog post ప్రతిసారీ site edit చేయక్కర్లేదు.
  CMS add చేయండి - non-technical person blog update చేయవచ్చు!

💡 TIP 8: Mobile First Design
  India లో 70%+ users mobile.
  Mobile view check first, then desktop.

💡 TIP 9: SEO Basics
  Every page: unique title + description.
  H1 tag: Main keyword include చేయండి.
  Image alt text: Describe images.

💡 TIP 10: Community Resources
  Framer Academy - free video tutorials.
  Framer Discord - help పొందవచ్చు.
  YouTube: "Framer tutorial" - Indian creators explain చేస్తున్నారు!
